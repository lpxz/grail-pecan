Jigsaw Introduction

This chapter of Jigsaw documentation is a quick overview of its architecture. It is recommended that you read these documents before proceeding any further.

  • Jigsaw architecture is a brief overview of the server's internal. It is meant to include only what is needed to run and configure the server.
  • The resource factory is a central piece of Jigsaw whose role is to build resource objects out of existing files or directories. This document provides a brief insight of how this mapping is done.
  • The installation page will explain you how to install and run Jigsaw.

Also available are two self-contained documents about:

  • The current Jigsaw design rationale.
  • Jigsaw performance evaluation.
