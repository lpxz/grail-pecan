Jigsaw tutorials

This section of the Jigsaw user's guide provides you three tutorials:

Configuring Jigsaw
Will explain you all the steps to configure Jigsaw once you have unpacked the distribution.
Writing new resources
Explains you how to write new resources to handle your own informations.
Writing new filters
Explains you how to write new resource filters, and what they can be used for.
Writing new server-side include commands
Explains how you can extend the very powerfull server-side include resource, by adding new commands, programmed in Java.
