LabelBureauResource

The LabelBureauResource implements a full-blown PICS label bureau. In PICS terminology, a label bureau is a network service that delivers labels rating internet documents.

The Label bureau configuration depends on the implementation of the label database you use. The only supported database, as of today, uses the file system to store labels (see the PICS administration guide for more informations).

Inherits

The LabelBureauResource class inherits from the following classes:

  • PostableResource

Attributes description

The LabelBureauResource defines the following attributes:

  • bureau

bureau
semantics
The identifier of the database that the bureau should query to answer requests. If you use the simple file-based version of the database, you should provide here the top-level directory of the bureau's database.
type
This attribute is an editable  FileAttribute.
default value
This attribute has no default value.
